Justify-content:

This property aligns flex items along the main axis of the flex container.

flex-start:

Items are packed at the start of the main axis.

flex-end:

Items are packed at the end of the main axis.

center:

Items are centered along the main axis.

Space-between:

Items are evenly distributed with the first item at the start and the last item at the end.

space-around:

Items are evenly distributed with equal space around them

space-evenly:

Items are evenly distributed with equal space around them, including at the start and end.

Align-items:

This property aligns flex items along the cross axis of the flex container.

flex-start:

Items are aligned at the start of the cross axis..

flex-end:

Items are aligned at the end of the cross axis.

Center:

IItems are centered along the cross axis.

baseline:

Items are aligned based on their baselines.

stretch:

Items are stretched to fill the container along the cross axis.

Gap:

This property sets the gap (space) between flex items. A length value or a percentage. It's a shorthand for row-gap and column-gap in Grid and a shorthand for gap in Flexbox.

Flex-direction:

This property establishes the main axis of the flex container and the direction in which flex items are placed.

Row:

Items are placed in the same direction as the main axis (left to right).

row-reverse:

Items are placed in the opposite direction of the main axis (right to left).

column:

Items are placed in the same direction as the cross axis (top to bottom).

column-reverse:

Items are placed in the opposite direction of the cross axis (bottom to top).

flex-wrap:

This property defines whether flex items are forced onto a single line or can wrap onto multiple lines.

nowrap:

All flex items are forced onto one line.

wrap:

Flex items wrap onto additional lines if needed..

wrap-reverse:

Flex items wrap onto additional lines in reverse order.

These properties collectively allow you to control the layout, alignment, and spacing of flex items within a flex container. By adjusting these properties, you can create a variety of flexible and responsive layouts.
